- React JS  
- Redux Toolkit
- Firebase 
- React-router
- MUI


## Installation

1. Clone the repo

```bash
 https://github.com/Emma177/Store-Management-App.git
```

2. Install NPM packages
```
npm install
```
3. Add .env file with your firebase config ??

4. Run the app
```
npm start
```
